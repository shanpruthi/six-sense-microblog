# six sense microblog at http://6ixsense.net

A Microblog that lets users post anonymously to a collective feed.

MEAN Stack App with Redis Caching and JWT Authentication

## Features

- JSON Web Tokens
- Bcrypt Storage Hashing
- Redis Caching
- Docker
- Session Storage

## Quick Start

```
npm install
bower install
node app.js
```
